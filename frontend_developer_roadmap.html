Frontend Development

Frontend development is the development of the graphical user interface of a website, through the use of HTML, CSS, & JavaScript, so that users can view & interact with that website.

  • HTML: The HyperText Markup Language, or HTML, is the standard markup language for documents designed to be displayed in a web browser.

  • CSS: Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language, such as HTML or XML.

  1. Pure CSS

  2. Bootstrap CSS (Framework)

  3. Tailwind CSS (Framework)

  4. Sass (Preprocessor)


  • JavaScript: JavaScript, often abbreviated JS, is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. It is used to add functionality to the website.

  1. Vanilla JS

  2. jQuery (Library)

  3. React (Framework)

  4. Angular (Framework)

  5. Vue (Framework)

  6. Typescript (Preprocessor)
